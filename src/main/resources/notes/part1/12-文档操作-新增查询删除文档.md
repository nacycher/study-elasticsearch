## 文档操作

### 新增文档
POST /索引库名/_doc/文档id  
需要加上文档id，否则会自动生成id

### 查询文档
GET /索引库名/_doc/文档id

### 删除文档
DELETE /索引库名/_doc/文档id

```kibana
# 新增文档
POST /heima/_doc/1
{
  "info":"黑马程序员java讲师",
  "email":"123@qq.com",
  "age":23,
  "name":{
    "firstName":"马",
    "lastName":"开山"
  }
}

# 查询文档
GET /heima/_doc/1

# 删除文档
DELETE /heima/_doc/1
```
