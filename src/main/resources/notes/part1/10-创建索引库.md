## 创建索引库
ES使用result风格的API请求方式  
PUT /索引库名

```kibana
# 创建索引库
PUT /heima
{
  "mappings": {
    "properties": {
      "info":{
        "type": "text",
        "analyzer": "ik_smart"
      },
      "email":{
        "type":"text",
        "index": false
      },
      "name":{
        "type": "object",
        "properties": {
          "firstName": {
            "type":"keyword"
          },
           "lastName": {
            "type":"keyword"
          }
        }
      }
    }
  }
}
```
